# 안녕하세요!

GitHub Pages와 Jekyll을 사용한 첫 번째 블로그 포스트입니다.

## 이 블로그에서 다룰 내용들

- 개발 경험 공유
- 기술 학습 내용
- 프로젝트 소개
- 코딩 팁과 트릭

## 코드 예제

Jekyll에서는 코드 하이라이팅도 지원합니다:

```javascript
function hello() {
    console.log("Hello, GitHub Pages!");
}

hello();
```

```python
def greet(name):
    return f"안녕하세요, {name}님!"

print(greet("개발자"))
```
